# Swati Deshmukh portfolio
A responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript** to showcase my skills, projects, and contact information.

This portfolio highlights my journey as a **Frontend Developer** and demonstrates my ability to build clean, responsive, and user-friendly web interfaces.

---

## 🚀 Features

* Responsive design for desktop, tablet, and mobile
* Modern UI with gradient styling
* Smooth scrolling navigation
* Animated hero section
* Interactive skill progress bars
* Projects showcase section
* Contact form with validation
* Scroll progress indicator
* Mobile navigation menu

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (DOM Manipulation)
